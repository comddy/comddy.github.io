字典

字典键–值关系

名称 唯一性 可存储数据类型 可变性
key(键) 唯一 数字、字符串、元组 不可变
value(值) 不唯一 任意 可变

字典常用方法

描述 方法
创建空列表 d = {} 或 d = dict()
插入元素 d[“key”] = value
更新元素 d[“key”] = new_value # 存在覆盖,不存在则插入
或 d.update({“a”:1,”b”,2}) # 同上
删除元素 del key[“key”] # 不返回值
或 d.pop(“key”) # 返回被删除键对应的值,不存在key均会报错
获取key值 d[“key”] # 不存在会报错
d.get(“key”) # 不存在会返回None
清空字典 d.clear() 或 d = {}
销毁字典 del d
插入获取结合 d.setdefault(key,value) # 存在key则返回key的值，不然插入key:value
获取字典items list(d.items()) # 返回列表，列表包含元组
获取字典keys list(d.keys())
删除字典最后的键值对 d.popitem() # 返回元组
获取字典values list(d.values())
判断键是否存在 key in d 或 key not in d

用字典格式化字符串

列表

列表常用方法

描述 方法
创建列表 t = [1,2,3]
取值 t[index]
取索引 t[value]
新增 t.append(value) # 在最后插入
t.insert(index,value) # 在index处插入
修改 t[index] = new_value
删除 del t[index] # 删除指定索引的数据
t.pop(index) # 删除并返回指定索引的数据
t.remove(value) # 删除list中第一个value
t.pop() # 删除并返回末尾的数据
合并 new_t = t + [1,2,3] # 合并成一个新list
t.extend([1,2.3]) # 将list追加到 t 的末尾（建议）
排序 # 就地排序(默认顺序),True逆序
t.sort(reverse=False)
t.sort(key=len) # key 接受一个函数，根据函数排序，例如len根据字符串长度顺排
sorted_t = sroted(t) # sorted排序任意序列
t.reverse() # 就地逆序
reversed(t) # 返回t逆序的迭代器
唯一有序列表 sorted_only_one = set(t.sort())
